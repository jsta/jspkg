# Demo of npm packaging

A demo of the steps to create a npm package

## Steps

* set npm configs

    * `npm set init.author.email <>`

    * `npm set init.author.name "jsta"`

    * `npm set init.license "GPL3"`

* `git init`

* create a minimal README

* npm init --yes
